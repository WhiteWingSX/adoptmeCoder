# Adoptme - Coder

Herramientas a utilizar:

 - Postman

### Como inicializar:

1. descarga el proyecto de GitHud utilizando git clone https://github.com/WhiteWingSX/adoptmeCoder.git
2. descarga los modulos de node `npm i`
3. inicia el proyecto `npm start` Este iniciara en el **locahost:8080**
4. Pudes realizar las pruebas a traves de los siguientes endpoints disponibles:
* GET: localhost:8080/api/mocks/mockingusers
* GET: localhost:8080/api/mocks/mockingpets
* POST: localhost:8080/api/mocks/generateData
